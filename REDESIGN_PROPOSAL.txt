  1. DIAGNOSTIC DE L'APPLICATION ACTUELLE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Points forts actuels :
  ✓ Dark theme cohérent (#09090b background, palette zinc)
  ✓ Logique fonctionnelle solide (tweaks, benchmark, nettoyage, overlay…)
  ✓ Toutes les données système sont bien récupérées et affichées
  ✓ Typographie correcte (Orbitron pour les labels, Inter, JetBrains Mono)

Points faibles identifiés :
  ✗ 10 onglets dans la sidebar → trop chargé, l'utilisateur est perdu
  ✗ HomeTab et MonitorTab font doublon — deux pages de stats qui se ressemblent
  ✗ Les graphiques (sparklines) sont minuscules : 56×16px, illisibles
  ✗ Pas de vraie hiérarchie visuelle : tout se ressemble, rien ne ressort
  ✗ Les jauges circulaires (RingProgress) sont petites et peu impactantes
  ✗ Les tweaks sont listés sans catégorie, c'est une longue liste monotone
  ✗ Le design "gaming" est timide : on peut aller bien plus loin côté style
  ✗ Pas d'accent de couleur fort ni de hero visuel
  ✗ Les stats temps-réel manquent de mise en valeur (petits chiffres)
  ✗ L'overlay est trop basique pour être vraiment utile en gaming
  ✗ Les pages Login/Register/Home ont un design complètement différent du Dashboard

ON PASSE DE 10 ONGLETS À 6 SECTIONS LOGIQUES :

  AVANT (confus)          →   APRÈS (clair)
  ─────────────────────────────────────────────────────────────────────
  1. Accueil                  1. DASHBOARD  (Accueil + Monitoring fusionnés)
  2. Monitoring               2. PERFORMANCE (Optimisations + Processus fusionnés)
  3. Processus                3. RÉSEAU     (Réseau → garde sa page)
  4. Réseau                   4. NETTOYAGE  (Nettoyage → garde sa page)
  5. Optimisations            5. JEUX       (Jeux + Benchmark fusionnés)
  6. Démarrage                6. SYSTÈME    (Démarrage + Paramètres fusionnés)
  7. Nettoyage
  8. Jeux
  9. Benchmarks
  10. Paramètres

POURQUOI CE REGROUPEMENT ?
  • Dashboard = une seule page qui dit TOUT sur l'état actuel du PC
  • Performance = tout ce qui touche à l'optimisation (tweaks + processus)
  • Réseau = une section dédiée, les gamers s'y intéressent beaucoup
  • Nettoyage = action spécifique, garde son espace
  • Jeux = bibliothèque + benchmark, deux trucs liés aux jeux
  • Système = tout ce qui est "admin" (démarrage + paramètres)

Concept : professionnel et gaming à la fois. Comme un HUD de cockpit de simulation.
Référence visuelle : MSI Center, HWiNFO64 moderne, NZXT CAM, mais en BIEN plus beau.

PALETTE DE COULEURS :
  Background principal  : #080810  (quasi noir avec teinte bleue/violette)
  Surface cards         : #0f0f1a  (légèrement plus clair)
  Surface elevated      : #161625  (cards sur fond)
  Borders               : #1e1e35  (bordure subtile)
  Borders active        : #2d2d55  (bordure hover/focus)

  Bleu accent (CPU)     : #3b82f6  → #60a5fa  (gradient)
  Violet (RAM)          : #7c3aed  → #a78bfa  (gradient)
  Orange/Rouge (Temp)   : #ea580c  → #fb923c  (gradient)
  Jaune (Disque)        : #d97706  → #fbbf24  (gradient)
  Vert (OK / Réseau)    : #16a34a  → #4ade80  (gradient)
  Violet clair (GPU)    : #9333ea  → #c084fc  (gradient)

  Texte principal       : #f0f0ff  (blanc légèrement bleuté)
  Texte secondaire      : #8888aa  (gris-bleu)
  Texte muté            : #44445a  (très discret)

TYPOGRAPHIE :
  Titre / Logo          : Orbitron Black, uppercase, letter-spacing 0.15em
  Valeurs numériques    : JetBrains Mono, toujours tabular-nums
  Labels / Corps        : Inter, pour tout le reste

BORDURES & ARRONDI :
  • Systématiquement border-radius: 0 sur les grandes cards (gaming flat)
  • Petits éléments (badges, boutons) : border-radius: 2px max
  • Jauges et cercles : border-radius: 9999px (ils sont ronds par définition)

EFFETS SPÉCIAUX (utilisés avec parcimonie) :
  • Une fine ligne colorée (2px) en haut des cards importantes = "accent bar"
  • Box-shadow très léger sur les cards élevées (pas de glow excessif)
  • Gradient de fond : très subtil radial gradient bleu-violet au centre de l'écran
  • Séparateurs : dégradé de couleur → transparent (pas de ligne full-width)

DÉTAIL DES JAUGES CIRCULAIRES :
  • Taille 96px avec épaisseur de trait de 6px
  • Valeur numérique en grand (24px, JetBrains Mono) au centre
  • Label en dessous (10px, gris)
  • Animation fluide sur changement de valeur
  • Couleur change du vert → orange → rouge selon seuil

DÉTAIL DES GRAPHIQUES (AREA CHARTS) :
  • Hauteur 80px minimum (vs 16px actuellement)
  • Area chart avec gradient de fill (couleur opaque en haut → transparent en bas)
  • Ligne de trace (2px) + zone remplie
  • Axe Y implicite (pas de labels, mais la grille est visible à 10% d'opacité)
  • 30 points = 1 minute d'historique à 2s d'intervalle

TOPBAR DU DASHBOARD :
  • Logo OPTIPC (Orbitron) à gauche
  • Score global de performance (grand chiffre coloré) au centre
  • Indicateur LIVE animé (point vert pulsant)
  • Nom du PC et OS à droite

  11. GRAPHIQUES TEMPS RÉEL — REFONTE COMPLÈTE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PROBLÈME ACTUEL :
  Les sparklines font 56×16px ou 60×28px. Impossible de lire une tendance.

NOUVEAU COMPOSANT "AreaChart" :
  • Dimensions : 100% de largeur × 80px de hauteur minimum
  • SVG avec path fill (area gradient) + path stroke (ligne)
  • Le fill : couleur à 20% d'opacité en haut → 0% en bas
  • La ligne : 2px, couleur pleine
  • Grille horizontale : 3 lignes à 33%, 66%, 100% (couleur 5% opacité)
  • Pas d'axes ni de labels (propre et compact)
  • Transition smooth : les points se décalent vers la gauche

POUR CHAQUE MÉTRIQUE :

  CPU (bleu #60a5fa) :
    • Jauge Ring 96px + Valeur en gros + AreaChart 100%×80px
    • Min/Max visible en hover
    • Threads actifs en sous-label

  RAM (violet #a78bfa) :
    • Pareil + GB utilisé / GB total affiché
    • Progression linéaire additionnelle

  Température (orange #fb923c) :
    • Couleur qui change automatiquement : vert si < 60°C, orange si < 80°C, rouge si ≥ 80°C
    • Animation "pulse" si température critique

  GPU (violet clair #c084fc) :
    • Usage % + Température GPU séparés avec 2 graphs
    • VRAM : barre de progression (X MB / Y MB)

  Ping (vert/jaune/rouge selon valeur) :
    • Historique des 30 dernières mesures (toutes les 5s = 2.5 min)
    • Valeur min/max/moy sur la fenêtre
    • Couleur dynamique

  Disque :
    • Pas de graphique temps réel (ça ne change pas assez vite)
    • Simplement une barre de remplissage + Go utilisés / Go total

  12. TWEAKS — REFONTE DÉTAIL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

GROUPES PROPOSÉS :

  ▶ PERFORMANCES (2 tweaks)
     • Plan haute performance   → active le plan d'alimentation "Haute performance"
     • Priorité CPU premier plan → WinSAT + MMCSS foreground boost

  ▶ GAMING (2 tweaks)
     • Désactiver Xbox Game Bar → HKCU\...\GameDVR disabled
     • Activer le Game Mode     → HKCU\...\GameMode enabled

  ▶ VISUEL & SERVICES (3 tweaks)
     • Désactiver les animations Windows → SystemParametersInfo
     • Désactiver SysMain (Superfetch)  → service stop + disable  ⚠ Admin
     • Désactiver Windows Search        → service stop + disable  ⚠ Admin

  ▶ RÉSEAU (1 tweak)
     • Optimiser TCP/IP         → netsh int tcp set global autotuninglevel=normal

DESIGN DES ROWS DE TWEAKS :
  • Icône à gauche avec couleur du groupe
  • Nom du tweak (bold) + description courte (gris)
  • Badge "⚠ Admin" si nécessaire (jaune)
  • Toggle à droite
  • Fond légèrement différent quand ON (plus clair)
  • Séparateurs visuels entre groupes (pas de border, juste espace + label groupe)
